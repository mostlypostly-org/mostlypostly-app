<decisions_made>

**Root Cause**: The onclick attributes on AI Generate buttons in vendorAdmin.js evaluate
`document.getElementById('some-id').value` INLINE in the HTML attribute string. When the browser
executes the onclick, if `getElementById` returns null, a TypeError is thrown BEFORE
`aiGenerateDesc` is ever called. The browser swallows this silently — no loading state,
no alert, "absolutely nothing" to the user.

User confirmed: (1) product name IS filled in, (2) button shows zero visual response (stays same).

**Fix Approach**: Change onclick attributes to pass element ID strings (not inline `.value` calls).
Update `aiGenerateDesc` to accept the product name input's ID as a string and do the lookup
internally with proper error handling.

**Scope**: Fix in 3 places in vendorAdmin.js:
1. Inline per-vendor form (line ~424): onclick for `aiGenerateDesc` inline form
2. Top-level form (line ~992): onclick for `aiGenerateDesc` top-level form
3. Brand detail page (line ~1786): onclick for `aiGenerateDescBrand`
The edit campaign page `aiGen` function was already fixed in commit 077230b — NO CHANGES NEEDED THERE.

**Implementation detail**:
```js
// OLD (silent TypeError if getElementById fails):
onclick="aiGenerateDesc(this, 'Aveda', document.getElementById('inline-prod-Aveda').value, 'inline-desc-Aveda')"

// NEW (IDs passed as strings, lookup done inside function):
onclick="aiGenerateDesc(this, 'Aveda', 'inline-prod-Aveda', 'inline-desc-Aveda')"
```

Updated `aiGenerateDesc` signature:
```js
async function aiGenerateDesc(btn, vendorArg, productNameInputId, targetId) {
  // vendorArg: try getElementById first (handles top-level form), else use as literal string
  var vendorEl = document.getElementById(vendorArg);
  var vendorName = vendorEl ? vendorEl.value.trim() : vendorArg;
  // productNameInputId: always an element ID
  var productEl = document.getElementById(productNameInputId);
  var productName = productEl ? productEl.value.trim() : '';
  if (!vendorName || !productName) {
    alert('Enter vendor name and product name first.');
    return;
  }
  var target = document.getElementById(targetId);
  if (!target) { alert('Form field not found. Please refresh and try again.'); return; }
  // ... rest of function unchanged
}
```

Top-level form onclick becomes:
```html
onclick="aiGenerateDesc(this, 'top-form-vendor-name', 'top-form-product-name', 'top-form-desc')"
```
